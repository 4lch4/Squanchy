Tron

Tron is a bot I started in December 2016 to learn how to create Discord chat bots. Currently, he has a few different "reaction" commands at the request of various friends and server members. To get a list, invite him to your server, and use the +help command.

  • Repo: GitHub
  • Began Development: December 2016
  • Service: Invite to Your Server

Hermes

Hermes is an idea I came up with to help me text through Discord since I use it so frequently and have been known not to reply to text messages for hours at a time when my phone is in the other room...

Without going into too much detail, there's an Android app you install on your phone that will forward your text messages to a Discord bot (Hermes) which then sends them to you so you can read/reply to the text.

  • Node.js Repo: GitHub
  • Android Repo: GitHub
  • Started: January 2017
  • Service: Invite Hermes to your Server

Gamgee

Gamgee is my personal PowerShell module that helps with managing the various tasks I have to do around the web. For example, uploading the latest versions of Hermes or Tron to their production servers, or updating the version of iPowerShell, and of course my favorite one, downloading the latest Mortys available from PocketMortys.
